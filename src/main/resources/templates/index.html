This is Inventorying App - spring-based web application for inventory, where you can add items, view, filter and export to PDF. Also, you can carry out two types of inventory: “sighted” and “blind” and then export the result to a PDF file. When you create a new account, your role is ADMIN, so you can create ten accounts with role USER, with limited functionality.

Default login/pass is test/1234.
